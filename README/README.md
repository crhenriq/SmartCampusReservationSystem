# Smart Campus Resource Reservation System

## Overview
This project is a console-based Smart Campus Resource Reservation System written in C++.  
It allows students to view and reserve shared campus resources such as study rooms and sports courts.  
Administrators can manage campus resources, and the system automatically prevents reservation conflicts.

The was our Final Project for CS 1C, and it emphasizes object-oriented design, file persistence, and teamwork.

---

## Features
- Supports multiple user types (Students and Administrators)
- Multiple resource types (Study Rooms and Sports Courts)
- Reservation creation, cancellation, and modification
- Automatic conflict detection for overlapping reservations
- Search resources by name or ID
- Filter resources by type
- File-based persistence using CSV files
